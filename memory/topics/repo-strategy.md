# 仓库策略

## 默认结构

- 单个 Skill 独立仓库：主开发源
- Skill 合集仓库：稳定分发副本

## 默认收录门槛

- 至少达到“可演示”或“可交付”
- 满足 `模板/合集收录标准.md`
- 有最小样例、边界说明和状态说明
