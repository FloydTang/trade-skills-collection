# 开源工具总表

这份文档是 GitHub 开源发现项目的唯一长期主表。

它合并了原来的观察池、落地候选、决策记录三种角色。

## 使用规则

- 每个项目只保留一条主记录
- 只记录后续判断会重复用到的信息
- 没有足够证据时，直接写 `不确定`
- 不把日报整段复制进来

## 状态说明

- `观察中`：值得继续看，但还不到落地判断阶段
- `待判断`：已经进入较强相关候选，等待决定怎么接
- `已结论`：已经形成当前阶段明确判断
