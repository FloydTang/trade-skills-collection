# Security Policy

## Supported Use

This project is intended for public-web research only.

## Reporting a Vulnerability

If you discover a security issue in the code, dependencies, or workflow, please report it privately through GitHub security reporting if enabled, or open a minimal public issue without disclosing exploit details.

## Scope Notes

- Do not use this project to collect private or unauthorized data
- Do not store real customer secrets in example files or issues
- Review generated outreach content before sending it externally
